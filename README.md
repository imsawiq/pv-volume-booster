# EN

Client-side Fabric mod for **Plasmo Voice** that unlocks higher volume values for both your microphone and other players.

Perfect if the default `200%` cap feels too low and you want much stronger voice volume control.


### ✨ Features

- **Boost player volume above 200%** - Raises the normal Plasmo Voice limit up to much higher values
- **Boost microphone input** - Lets you increase your own microphone volume far beyond the default cap
- **Client-side only** - Install it on your client and use it without requiring server-side changes


### 📦 Dependencies

- **Fabric API** - Required
- **Plasmo Voice** - Required

# RU

Клиентский Fabric-мод для **Plasmo Voice**, который снимает стандартный лимит громкости как для твоего микрофона, так и для других игроков.

Подходит, если стандартных `200%` в Plasmo Voice просто не хватает и нужен реально более сильный буст громкости.


### ✨ Особенности

- **Повышение громкости игроков выше 200%** - Поднимает обычный лимит Plasmo Voice до гораздо больших значений
- **Повышение громкости микрофона** - Позволяет усиливать свой микрофон сильно выше стандартного предела
- **Полностью клиентский мод** - Ставится только на клиент и не требует изменений на стороне сервера


### 📦 Зависимости

- **Fabric API** - Обязательно
- **Plasmo Voice** - Обязательно
